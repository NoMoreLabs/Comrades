# Rarity Analysis

This folder contains rarity analysis, attribute count breakdowns, and trait statistics for both the Call Data Comrades (CDC) and Comrades of the Dead (COTD) collections on Ethscriptions.

## Naming Convention

- Files prefixed with **`cdc-`** relate to the **Call Data Comrades** collection (9,959 items)
- Files prefixed with **`cotd-`** relate to the **Comrades of the Dead** collection (666 items)

## Contents

### Call Data Comrades (CDC)
- **cdc-rarity.md**  
  Documentation on rarity methodology and findings for Call Data Comrades.

- **cdc-attribute-count-analysis.md**  
  Call Data Comrades grouped by their attribute count, with lists for each group.

- **cdc-attribute-synergy.json**  
  Data on attribute synergies within the Call Data Comrades collection.

- **cdc-rarity-grid.png**  
  Visual grid showing rarity distribution for Call Data Comrades.

- **cdc-rarity-ranks.csv** / **cdc-rarity-ranks.txt**  
  Rarity rankings for all Call Data Comrades.

- **cdc-traits-stats.md**  
  Detailed trait rarity statistics for Call Data Comrades, including frequency and rarity scores for all trait values.

### Comrades of the Dead (COTD)
- **cotd-rarity-rankings.txt**  
  Rarity rankings for all Comrades of the Dead.

- **cotd-rarity-statistics.txt**  
  Detailed trait rarity statistics for Comrades of the Dead, including frequency and rarity scores for all trait values.

## Usage

- Reference these files to analyze rarity, attribute distributions, and trait frequencies within each collection.
- Use the CSV/TXT files for programmatic access to rarity ranks.
- Visual assets can be used for presentations or further analysis.

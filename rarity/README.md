# Rarity Analysis

This folder contains rarity analysis, attribute count breakdowns, and trait statistics for the Call Data Comrades collection on Ethscriptions.

## Contents

- **comrade-rarity.md**  
  Documentation on rarity methodology and findings.

- **comrades-attribute-count-analysis.md**  
  Comrades grouped by their attribute count, with lists for each group.

- **comrades-attribute-synergy.json**  
  Data on attribute synergies within the collection.

- **comrades-rarity-grid.png**  
  Visual grid showing rarity distribution.

- **comrades-rarity-ranks.csv** / **comrades-rarity-ranks.txt**  
  Rarity rankings for all comrades.

- **comrades-traits-stats.md**  
  Detailed trait rarity statistics, including frequency and rarity scores for all trait values.

## Usage

- Reference these files to analyze rarity, attribute distributions, and trait frequencies within the Call Data Comrades collection.
- Use the CSV/TXT files for programmatic access to rarity ranks.
- Visual assets can be used for presentations or further analysis.
